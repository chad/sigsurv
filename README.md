# Sigsurv

This is a stupid implementation of Ward Cunningham's clever [Signature Survey](http://c2.com/doc/SignatureSurvey/).


## Installation

install it as:

    $ gem install sigsurv

## Usage

$ cd yourapp
$ sigsurv
